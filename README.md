# FinSight AI

## 🚀 Overview

FinSight AI converts complex financial news into simple, actionable insights tailored for different types of investors.

## 💡 Features

- News summarization
- Market sentiment analysis
- Actionable insights
- Personalized responses (Beginner, Intermediate, Senior)
- Domain guardrails (only financial news)

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API (LLM)

## ▶️ How to Run

1. Clone repo
2. Install dependencies:
   pip install -r requirements.txt
3. Add your API key in `.env`
4. Run:
   streamlit run app.py

## 📊 Example Use Case

Paste financial news → Get insights instantly

## ⚠️ Note

This tool provides general insights, not financial advice.
So, no financial decisions should be made considering the output of this tool.
